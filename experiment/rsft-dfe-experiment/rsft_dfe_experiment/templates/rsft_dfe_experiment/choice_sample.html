Learning Probabilities

First, you learn probabilities. The probabilities with which the option return points are hidden. The amount of points that each option offers are shown, such as 3 and 9. A click on one option causes the option to draw one the amounts, that it offers, accourding to the hidden probability associated with the amount. For instance, if the hidden probability for 3 points was 50%, the amount 3 would appear in half of the clicks, if you would do an infinite number of clicks. The probabilities sum up to 1 for each option.
